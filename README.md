# Capstone Project

## Description
This is a Python-based Capstone Project developed to demonstrate practical application of programming concepts and software development principles.

The project provides core functionality through Python modules and includes automated testing to verify correctness and reliability.

---

## Features
- Core project functionality
- User interaction
- Error handling
- Modular code structure
- Automated testing

---

## Requirements

Before running the project, install:

- Python 3.10 or higher

Check version:

```bash
python --version
```

---

## Installation

Clone or download the project.

Move into the project folder:

```bash
cd capstone-project
```

Install dependencies (if required):

```bash
pip install -r requirements.txt
```

---

## How to Run the Project

Run the main application:

```bash
python main.py
```

If using Google Colab:

1. Upload all project files
2. Open the notebook or script
3. Click **Runtime → Run All**

---

## Running Tests

Run all tests:

```bash
python -m unittest
```

Or run a specific test file:

```bash
python -m unittest test_project.py
```

Expected output:

```text
Ran X tests

OK
```
